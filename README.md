# 📝 MERN Blog Application

A full-stack **Blog Application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This application allows users to **create, read, update, and delete blogs**, upload images, and manage authentication securely using **JWT**.

The project follows a **clean folder structure, secure backend APIs, and full frontend-backend integration**.

---

# 🚀 Features

✅ Blog CRUD Operations (Create, Read, Update, Delete)  
✅ User Authentication with JWT  
✅ Upload Blog Images & Author Images  
✅ Image Storage using **Cloudinary**  
✅ Secure API using Middleware  
✅ Best Folder Structure & Backend Setup  
✅ Full Frontend & Backend Integration  

---
